## Description
- This is a running iOS application 
- It tracks your running path from your starting point to your ending point
- It tracks the distance (Miles and Kilometers)
- It tracks your running time 
- UI with Storyboard
- MVC Design Pattern

## Frameworks:
- Foundation
- UIKit
- CoreLocation
- MapKit

## What I've Learned:
